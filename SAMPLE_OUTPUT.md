
## Usage

1. Launch the application
2. Follow the on-screen prompts to:
   - Enter the temperature value
   - Select the source temperature scale
   - Select the target temperature scale
3. View the converted result
4. Choose to perform another conversion or exit

### Example Usage
```
Welcome to Temperature Converter!

Enter temperature value: 100
Select source scale:
1. Celsius
2. Fahrenheit
3. Kelvin
Choice: 1

Select target scale:
1. Celsius
2. Fahrenheit
3. Kelvin
Choice: 2

Result: 100°C = 212°F
```

## Temperature Conversion Formulas

The application uses the following standard conversion formulas:

**Celsius to Fahrenheit:**
```
°F = (°C × 9/5) + 32
```

**Celsius to Kelvin:**
```
K = °C + 273.15
```

**Fahrenheit to Celsius:**
```
°C = (°F - 32) × 5/9
```

**Fahrenheit to Kelvin:**
```
K = (°F - 32) × 5/9 + 273.15
```

**Kelvin to Celsius:**
```
°C = K - 273.15
```

**Kelvin to Fahrenheit:**
```
°F = (K - 273.15) × 9/5 + 32
```
